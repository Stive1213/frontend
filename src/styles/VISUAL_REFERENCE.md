## üìê Spacing & Typography

### Consistent Across Both Modes
- **Padding**: 1rem (16px) base, 1.5rem (24px) for cards, 0.75rem (12px) for compact
- **Border Radius**: 0.5rem (8px) standard, 0.75rem (12px) for cards, 1rem (16px) for modals
- **Font Sizes**: Consistent hierarchy (text-sm, text-base, text-lg, text-xl, text-2xl)
- **Font Weights**: Regular (400), Medium (500), Semibold (600), Bold (700)

When switching themes:
- **Duration**: 0.2s (200ms)
- **Easing**: ease (smooth)
- **Properties**: background-color, border-color, color
- **Result**: Smooth, professional transition without jarring changes

## üéØ Implementation Notes

1. **Always test both modes** - Visual appearance should match these descriptions
2. **Use tokens, not hardcoded colors** - Ensures consistency
3. **Check contrast ratios** - Verify accessibility in both modes
4. **Apply glows sparingly** - Only in dark mode, only for emphasis
5. **Image overlays** - Essential in dark mode to prevent visual breaks
6. **Shadow depth** - Darker in dark mode for better depth perception

---

This visual reference should guide all UI implementation to ensure pixel-perfect consistency across both light and dark modes.
